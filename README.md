## ⚡ Filosofía de Ingeniería (Core Values)

* **Planificación Rigurosa:** Escribo diagramas de flujo y defino contratos de APIs antes de tirar la primera línea de código. Un buen software se esculpe con diseño arquitectónico previo.
* **Tipado Estricto & Robustez:** Adoro TypeScript. El tipado no es una restricción; es una póliza de seguridad que auto-documenta el sistema y facilita refactorizaciones sin miedo.
* **Perfilado y Rendimiento:** Si el hilo principal del navegador se bloquea, es un bug. Optimizo el tamaño de los bundles, minimizo el Cumulative Layout Shift (CLS) y aprovecho la GPU del cliente mediante shaders cuando es necesario.
* **Alineación de Negocio:** No programo para añadir líneas de código; programo para resolver problemas reales. Cada decisión de caching local, pre-renderizado e indexación se traduce en mejores costos de infraestructura y mayor conversión.
