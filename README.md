\## ⚡ Filosofía de Ingeniería (Core Values)



\*   \*\*Planificación Rigurosa:\*\* Escribo diagramas de flujo y defino contratos de APIs antes de tirar la primera línea de código. Un buen software se esculpe con diseño arquitectónico.

\*   \*\*Tipado Estricto \& Robustez:\*\* Adoro TypeScript. El tipado no es una restricción; es una póliza de seguridad que auto-documenta el sistema y facilita refactorizaciones sin miedo.

\*   \*\*Perfilado y Rendimiento:\*\* Si el hilo principal del navegador se bloquea, es un bug. Optimizo el tamaño de los bundles, minimizo el Cumulative Layout Shift (CLS) y aprovecho la GPU del cliente con shaders matemáticos cuando es necesario.

\*   \*\*Alineación de Negocio:\*\* No programo para añadir líneas de código; programo para resolver problemas. Cada decisión de diseño técnico (caching local, Edge render) se traduce en reducción de latencia y mejor conversión SEO.
